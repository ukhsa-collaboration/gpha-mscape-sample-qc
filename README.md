# mscape-sample-qc

A repository containing code relating to per sample quality control of samples submitted to mscape.

## Installation

Clone repo:
```
git clone git@github.com:ukhsa-collaboration/mscape-sample-qc.git
cd mscape-sample-qc
conda create -n mscape_sample_qc python
conda activate mscape_sample_qc
```

User installation:
`pip install .`

For developers:
`pip install -e ".[dev]"`
